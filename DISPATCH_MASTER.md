### ФАЗА 2: Форк Gonka (неделя 1–2)

**Промпт 2.1 — Клонировать Gonka:**
```
Открой терминал и выполни:

cd ~/Razum/fork
git clone https://github.com/gonka-ai/gonka.git
git clone https://github.com/gonka-ai/gonka-openai.git

Затем изучи структуру обоих репозиториев.
Создай файл fork/ANALYSIS.md с описанием:
- Все папки и файлы
- Какие технологии используются
- Какие зависимости нужно установить
- Как запустить локально
- Все места где упоминается "Gonka" или "GNK"
```

**Промпт 2.2 — Ребрендинг:**
```
В папке fork/gonka замени во всех файлах:
- "Gonka" → "Razum"
- "gonka" → "razum"  
- "GNK" → "RZM"
- "gnk" → "rzm"

Не трогай внутреннюю логику и бизнес-код. Только строки, имена, комментарии, конфигурации.
Создай fork/REBRAND_LOG.md с логом всех изменений.
Убедись что код компилируется после замен.
```

**Промпт 2.3 — Попробовать запустить:**
```
Попробуй запустить локальную тестовую сеть Razum из папки fork/gonka.
Следуй инструкциям из fork/ANALYSIS.md.
Если нужны зависимости (Go, Docker) — установи их.
Если что-то не работает — запиши ошибки в fork/ISSUES.md.
Если всё работает — запиши инструкцию в fork/HOW_TO_RUN.md.
```

### ФАЗА 3: Веб-приложение (неделя 3–5)

**Промпт 3.1 — Создать веб-приложение:**
```
Создай Next.js приложение в папке ~/Razum/web-app:

npx create-next-app@latest razum-web --typescript --tailwind --app

Создай страницы:
- / (лендинг — возьми за основу website/index.html, конвертируй в React)
- /chat (чат с AI — стриминг текста посимвольно)
- /models (список доступных моделей: Llama 3, Mistral, Stable Diffusion)
- /pricing (планы подписки: Free $0, Basic $20, Pro $50)
- /login и /register (регистрация через email)
- /account (профиль, баланс кредитов, история)

Стиль: тёмная тема, фон #06080d, акцент #00e59b, шрифт Instrument Sans.
Адаптивный дизайн (мобильный).
```

**Промпт 3.2 — Подключить API для чата:**
```
В web-app подключи чат к API. На первом этапе используй OpenAI-совместимый API.

Создай файл web-app/lib/api.ts:
- Функция sendMessage(prompt, model) → стрим ответа
- Поддержка Server-Sent Events для стриминга
- Выбор модели (Llama, Mistral)
- Обработка ошибок

Для тестирования можно использовать любой OpenAI-совместимый эндпоинт.
Когда будет своя нода Razum — просто поменяем URL.
```

**Промпт 3.3 — Добавить Stripe:**
```
В web-app добавь оплату через Stripe:

npm install stripe @stripe/stripe-js

Создай:
- app/api/stripe/checkout/route.ts — создание checkout session
- app/api/stripe/webhook/route.ts — обработка webhook
- app/pricing/page.tsx — страница с планами и кнопками оплаты

Планы:
- Free: 50 запросов/день, $0
- Basic: 2000 запросов/день, $20/мес
- Pro: безлимит + API, $50/мес

Используй Stripe test mode ключи (pk_test_... и sk_test_...).
Я введу реальные ключи потом.
```

**Промпт 4.1 — Создать мобильное приложение:**
```
Создай React Native приложение в папке ~/Razum/mobile-app:

Стиль: тёмная тема, акцент #00e59b.
Подключи тот же API, что и в веб-приложении.
```

**Промпт 4.2 — Подготовить к публикации:**
```
Подготовь мобильное приложение к публикации:

1. Создай иконку приложения (1024x1024) и splash screen
2. Настрой app.json (название "Razum AI", bundleId "ai.razum.app")
3. Собери APK для Android: eas build --platform android
4. Собери для iOS: eas build --platform ios
5. Создай описание для App Store и Google Play
6. Сделай скриншоты (5 штук)

Запиши инструкцию в mobile-app/PUBLISH.md
```
