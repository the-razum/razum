# Razum AI — Пошаговая инструкция по запуску

## Шаг 1: Опубликовать сайт на Netlify (5 минут)

1. Открой https://app.netlify.com/drop
2. Перетащи папку `website/` в окно браузера
3. Через 10 секунд получишь ссылку
4. Зарегистрируйся на Netlify (бесплатно)
5. Переименуй сайт: Site settings → Change site name → `the-razum`
6. Итоговая ссылка: `https://the-razum.netlify.app`
7. Запиши ссылку в `docs/LINKS.md`

## Шаг 2: Создать Telegram-канал (5 минут)

1. Открой Telegram → Создать канал
2. Название: **Razum AI**
3. Юзернейм: **@RazumAI** (или @Razum_AI если занято)
4. Описание — скопируй из `marketing/social_media_launch.md`
5. Опубликуй первый пост — скопируй из того же файла
6. Замени `[ВСТАВЬ ССЫЛКУ]` на ссылку с Netlify
7. Запиши ссылку канала в `docs/LINKS.md`

## Шаг 3: Создать Twitter/X (5 минут)

1. Зайди на x.com → зарегистрируйся
2. Юзернейм: **@RazumAI**
3. Био и первый твит — из `marketing/social_media_launch.md`
4. Запиши ссылку в `docs/LINKS.md`

## Шаг 4: Создать GitHub (5 минут)

1. Зайди на github.com → создай организацию **the-razum**
2. Создай репозиторий **razum** (публичный)
3. README — из `marketing/social_media_launch.md` (раздел GitHub)
4. Запиши ссылку в `docs/LINKS.md`

## Шаг 5: Обновить ссылки на сайте

После создания всех аккаунтов — обнови ссылки в `website/index.html`:
- В футере замени `href="#"` на реальные ссылки Twitter, Telegram, GitHub
- Перезалей папку website на Netlify
