# Razum AI

> Разум, который принадлежит тебе.

Децентрализованная сеть AI-вычислений. Форк Gonka AI с фокусом на СНГ-рынок и простоту для обычного пользователя.

## Что это

Razum — сеть, где GPU по всему миру выполняют реальные AI-задачи (инференс, тренировка, файнтюнинг), а участники получают за это токены RZM. Для пользователя это выглядит как ChatGPT с подпиской картой — никакого крипто на поверхности.

## Двухслойная архитектура

**"Зал" (снаружи)** — обычный SaaS: сайт, приложение, подписка через ЮKassa/Stripe, ООО, расчётный счёт. Как Netflix.

**"Кухня" (внутри)** — децентрализованная GPU-сеть: блокчейн, ноды, токен RZM, верификация PoUC. Как Bitcoin, но для AI.

Пользователь видит только "зал". Майнер видит "кухню".

## 6 козырей Razum

1. **Без цензуры** — open-source модели без корпоративных фильтров
2. **Не отключат** — децентрализованная сеть, нет сервера который можно выключить
3. **Данные — твои** — ноды не хранят историю, нет центральной базы
4. **Оплата из России** — ЮKassa, МИР, СБП, за 10 секунд
5. **Зарабатывай** — подключи GPU = получай RZM
6. **Выбор моделей** — Llama, Mistral, Qwen, Stable Diffusion, Whisper

## Поэтапный запуск

### Месяц 1–2: Обычный веб-сервис (без блокчейна)
Сайт с чатом (Next.js) + ЮKassa/Stripe + 1–2 арендованных GPU (Vast.ai)

### Месяц 3–4: Подключаем майнеров
Координатор задач (Go + Redis) + Electron-майнер + внутренний токен RZM (пока в БД)

### Месяц 5–8: Добавляем блокчейн
Форк Gonka (Cosmos SDK) + настоящие кошельки + DEX (Osmosis)

### Месяц 9–12: Масштабирование
Листинг на CEX (MEXC, Gate.io) + мобильные приложения + реферальная программа

## Юридическая структура

ООО на УСН 6%. ОКВЭД: 62.01 + 63.11. ЮKassa для платежей. Пользователь платит рублями, токен — для майнеров и крипто-рынка.
