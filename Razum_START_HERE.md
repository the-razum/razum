## СЕГОДНЯ (30–60 минут)

### Шаг 1: Застолби название везде (15 мин)

- [ ] Telegram: создай канал → t.me/RazumAI
      Настройки: название "Razum AI", описание "Децентрализованный AI для всех. Вычисления принадлежат разуму."
- [ ] Twitter/X: зарегистрируй @RazumAI
- [ ] GitHub: создай аккаунт github.com/the-razum
- [ ] Discord: создай сервер "Razum AI"

Если @RazumAI занят — пробуй @Razum_AI, @RazumNetwork, @RazumChain.

### Шаг 2: Опубликуй сайт (5 мин)

1. Скачай файл index.html (уже с брендом Razum)
2. Создай папку "razum" на рабочем столе
3. Положи туда index.html
4. Открой в браузере: app.netlify.com/drop
5. Перетащи папку "razum" в окно браузера
6. Через 10 секунд получишь ссылку (например: the-razum.netlify.app)
7. Зарегистрируйся на Netlify → переименуй сайт в the-razum
