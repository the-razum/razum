# Razum AI — Дорожная карта запуска рабочей сети

## Путь от кода до "люди майнят RZM"

Вот реалистичный план из 6 этапов. Каждый этап — конкретные действия, сроки и бюджет.

---

## ЭТАП 1: Локальная сборка (1–2 дня, $0)

**Что делаем:** форкаем код, ребрендим, собираем и запускаем на своём компьютере.

**Требования:** твой ПК с Linux/macOS/WSL2, Go 1.21+, Docker.

### Шаги:
```bash
# 1. Клонировать
cd ~/Razum/fork
git clone https://github.com/gonka-ai/gonka.git
git clone https://github.com/gonka-ai/gonka-openai.git

# 2. Ребрендинг (скрипт уже готов)
cd gonka
bash ../rebrand.sh

# 3. Установить Go (если нет)
# https://go.dev/dl/ → скачай и установи

# 4. Собрать бинарник
go mod tidy
make build
# Результат: бинарник ./razumd

# 5. Запустить локальную тестовую ноду
./razumd init my-node --chain-id razum-testnet-1
./razumd keys add validator
./razumd genesis add-genesis-account validator 1000000000urzm
./razumd genesis gentx validator 100000000urzm --chain-id razum-testnet-1
./razumd genesis collect-gentxs
./razumd start
```

**Результат:** блокчейн Razum крутится на твоём компьютере. Блоки создаются, токены RZM существуют. Но это только локально.

## ЭТАП 2: Тестнет на сервере (1 неделя, ~$30–60/мес)

**Что делаем:** поднимаем настоящий тестнет в интернете, чтобы другие люди могли подключиться.

### Что арендовать:

| Компонент | Где | Цена/мес |
|-----------|-----|----------|
| Chain + API Node | Любой VPS (Hetzner, DigitalOcean) | ~$20–40/мес |
| ML Node (GPU) | Vast.ai (RTX 3060) | ~$60/мес |

Рекомендую **Hetzner** — дёшево и надёжно для Chain Node. Не нужен GPU для блокчейна — GPU нужен только для ML Node.

### Шаги:
1. Арендовать VPS: 4 CPU, 8 ГБ RAM, 100 ГБ SSD (~$20/мес на Hetzner)
2. Установить Docker + Docker Compose
3. Залить код Razum через GitHub
4. Запустить Chain Node + API Node через docker-compose
5. Настроить публичные RPC/API эндпоинты
6. Арендовать GPU-сервер на Vast.ai ($0.03–0.08/час)
7. Подключить ML Node к Chain Node
8. Протестировать инференс через razum-openai SDK

**Результат:** рабочий тестнет. Можно отправлять AI-запросы и получать ответы. Токены тестовые (не имеют ценности).

## ЭТАП 3: Публичный тестнет + первые ноды (2–4 недели, ~$100–200/мес)

**Что делаем:** приглашаем людей подключать свои GPU и тестировать сеть.

### Шаги:
1. Написать документацию для нод-операторов (как подключиться)
2. Создать Docker-образ ML-ноды и залить на Docker Hub
3. Написать скрипт установки для нод-операторов (1 команда)
4. Поднять 2–3 seed-ноды в разных регионах (~$60/мес каждая)
5. Создать Faucet — раздатчик тестовых RZM (чтобы ноды могли стейкать)
6. Запустить Block Explorer (можно форкнуть Mintscan или использовать ping.pub)
7. Анонсировать в Telegram/Twitter: "Тестнет открыт, подключай GPU"

### Что нужно подготовить для нод-операторов:
```bash
# Простая установка для владельцев GPU
curl -sSL https://razum.network/install.sh | bash
# Скрипт скачивает Docker, настраивает и запускает ML-ноду
```

**Результат:** 10–50 нод от энтузиастов тестируют сеть. Баги находятся и фиксятся.

## ЭТАП 4: Аудит + подготовка к мейннету (4–8 недель, $3,000–15,000)

**Что делаем:** проверяем безопасность кода и готовимся к запуску с реальными деньгами.

### Обязательно:
1. **Аудит смарт-контрактов / модулей** — $3,000–10,000
   - Можно найти на Kwork или через крипто-аудиторов
   - Минимум: аудит модулей collateral, compute, rewards
   - Компании: Halborn, Trail of Bits (дорого) или фрилансеры (дешевле)

2. **Stress-testing** — бесплатно
   - Нагрузочное тестирование: 100+ одновременных запросов
   - Проверка слэшинга при невалидных результатах
   - Тест на Byzantine fault (ноды пытаются жульничать)

3. **Genesis файл** — бесплатно
   - Распределение начальных токенов (как в токеномике)
   - Список генезис-валидаторов (те, кто тестил на тестнете)
   - Параметры сети: block time, reward schedule, slashing params

### Опционально (но крайне желательно):
- Юридическая консультация по токену RZM: $500–2,000
  - Нужно ли регистрировать? В какой юрисдикции?
  - Налоговые последствия для нод-операторов

**Результат:** код проверен, тестнет стабилен, genesis готов.

## ЭТАП 5: Запуск мейннета (1 день, ~$200–500/мес на инфру)

**Что делаем:** запускаем реальную сеть, где RZM имеет ценность.

### Шаги:
1. Опубликовать genesis.json
2. Собрать начальный набор валидаторов (минимум 5–10)
3. Скоординировать запуск (все стартуют одновременно)
4. Запустить:
   - 3+ seed-ноды в разных регионах
   - Block Explorer
   - Faucet (опционально, для промо)
   - RPC/API эндпоинты

### Минимальная инфраструктура мейннета:

| Компонент | Кол-во | Цена/мес |
|-----------|--------|----------|
| Seed Nodes (VPS) | 3 | $60–120 |
| API Gateway | 1 | $20–40 |
| Block Explorer | 1 | $20 |
| ML Nodes (свои, для bootstrap) | 2 | $120–300 |
| **Итого** | | **$220–480/мес** |

### Процедура запуска:
```
День запуска:
00:00 — Публикуем genesis.json в Telegram + GitHub
00:00 — Валидаторы запускают ноды с genesis
00:05 — Первый блок создан → сеть жива!
00:10 — Публикуем пост "Razum mainnet is LIVE"
01:00 — Первые ML-ноды подключаются
01:00 — Первый AI-запрос через сеть → первые RZM заработаны
```

**Результат:** рабочая сеть. Люди подключают GPU и реально зарабатывают RZM.

## ЭТАП 6: Листинг + ликвидность (после стабилизации, 1–3 месяца)

**Что делаем:** даём RZM реальную стоимость.

### Шаги:
1. Листинг на DEX (Osmosis — стандарт для Cosmos-токенов)
   - Бесплатно через IBC-протокол
   - Создать пул RZM/OSMO или RZM/USDC

2. Листинг на мелких CEX (MEXC, Gate.io)
   - $5,000–50,000 листинг-фи (зависит от биржи)
   - Или бесплатно, если биржа заинтересована

3. CoinGecko + CoinMarketCap
   - Заявка бесплатно, нужен работающий блокчейн + ликвидность

**Результат:** RZM торгуется на бирже, нод-операторы могут продавать заработанные токены.
