# Razum

Decentralized AI compute network powered by community GPUs.

## What is Razum?

Razum connects thousands of GPUs worldwide into a single AI compute network. Users pay with a card (from $0/mo), GPU owners earn RZM tokens.

Built on Cosmos SDK with Proof-of-Useful-Compute consensus.

## Key Features

- **Card subscriptions** — no crypto wallet needed for users
- **Chat UI** — ChatGPT-like interface
- **Mobile apps** — iOS & Android
- **AI Marketplace** — deploy and monetize models
- **Geographic clustering** — low latency via nearest node routing

- Total supply: 1,000,000,000
- Consensus: Proof-of-Useful-Compute
